I learned a lot about programming by working with Flash's Actionscript. Here you can see some of the games I created, which are for some reason still available online.

The Maze Aventure

A simple game in which you had to guide a ball through a maze in a 2D environment. It had several obstacles like walls, boosts, mines, locked doors and even flying aliens.

Piano Avoider

A very simple game I created for a mobile game contest in which you are controlling a person that had to run around and avoid falling pianos.

The Maze Aventure 2

A total remake of The Maze Aventure 1 with more levels, better gameplay and fancier graphics. The game was written in such a way that it would be possible for players to create their own levels, submit them, and share them with other people to play. Unfortunately, I never got around to implementing this.
