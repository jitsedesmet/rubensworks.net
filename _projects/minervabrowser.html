Ghent University, at which I studied, uses a digital learning platform called Minerva. Initially, Minerva was not mobile-friendly. That is why I created an Android application for it, which is currently not being maintained anymore.

The application was built on top of the Minerva API which I created as well. The API scrapes the Minerva website and makes it available through an XML interface.

The application is able to consume the API using the Minerva Android Library.
