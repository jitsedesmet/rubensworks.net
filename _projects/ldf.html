--- layout: project title: "Linked Data Fragments" start: "August 2015" description: "Research on Linked Data interfaces to achieve Web-scale querying" weight: 3 ---

Linked Data Fragments is a research area where different kinds of Linked Data interfaces are investigated. As multiple Linked Data interfaces exist, different techniques are needed to achieve Web-scale querying over one or more of these interfaces. Good Linked Data interfaces for data publishers depend on a range of factors, and the right choice is a matter of trade-offs.

Comunica is a query engine that has been designed based on the Linked Data Fragments concept.
